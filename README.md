A fork of DialogueNodes by [NagiDev](https://github.com/nagidev/DialogueNodes).

####
## Base Features 
- Simple editor, straight-forward dialogue box
- Dialogue animations with bbcodes
- Conditional dialogues & options
- Support for variables & signals
- Character portraits & colors
- In-editor dialogue previewer
- Localization through `tres` files

## Extra Features
- Added support for global and local variables
- Story variables can be accessed from anywhere in your project using the StoryManager singleton
